
Create alias
$ git config alias.lg 'log --graph --online --decorate --all'

Check Repo History
$ git log
	options:
	--graph // branch graph
	--oneline
	--decorate 
	-p	// changes
	-1	// number of lines

Project Status
$ git status 

Add file to stage
$ git add

Commit file
$ git commit -m "message" 

Setup Git
$ git config --global user.name "<name>"
$ git config --global user.email <email>

Initialize git repo
$ git init
