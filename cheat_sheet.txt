random stuff 3

Change to branch
$ git checkout <branchname>

Create branch
$ git branch <branchname>


Reset local changes
$ git checkout
	options:
	-- <filename> // reset local change on file

Reset the stage
$ git reset HEAD
	options
	<filename> // unstaged file

Remove commit
$ git reset <commit> // commit ID

	options:
	--hard // remove changes

Amend last commit
$ git commit -amend

Differences
$ git diff
	options:
	$ git diff  --staged // changes between stage and last commit


Create alias
$ git config alias.lg 'log --graph --online --decorate --all'

Check Repo History
$ git log
	options:
	--graph // branch graph
	--oneline
	--decorate 
	-p	// changes
	-1	// number of lines

Project Status
$ git status 

Add file to stage
$ git add

Commit file
$ git commit -m "message" 

Setup Git
$ git config --global user.name "<name>"
$ git config --global user.email <email>

Initialize git repo
$ git init
